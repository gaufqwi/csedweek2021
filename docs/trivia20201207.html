  1. The Imitation Game is a movie starring Benedict Cumberbatch about what British computer scientist? The name comes from this man's idea of testing if a computer program has artificial intelligence by seeing if it can imitate a person in a conversation.

    Answer: Alan Turing

    Alan Turing is widely considered the father of computer science. His work breaking codes during World War II probably saved many lives.   2. What computer language that was invented in 1967 allows you to draw pictures by giving commands like FORWARD 50 and RIGHT 90 to a pen-carrying turtle?

    Answer: LOGO

    LOGO was designed by Wally Feurzeig, Seymour Papert, and Cynthia Solomon. Early turtles were often robots that rolled on the floor and drew on paper, but now most versions of LOGO use an on-screen turtle.

  3. What short-range wireless technology that is often used to connect mobile phones to other devices is named for a 10th century Danish king?

    Answer: Bluetooth

    Bluetooth unites phones, computers, speakers, and other devices the same way that King Harald Bluetooth united the various tribes of Denmark. At least that's what the people who came up with the name say.

  4. What number does the following Python code snippet print? 
    
tot = 0
for n in [1,2,3,4,5]:
    tot = tot + n
print(tot)

    Answer: 15

    The loop repeats five times, with the variable n taking the values 1, 2, 3, 4, and 5. Each time the value of n is added to tot, which starts at 0. So in the end tot equals 0 + 1 + 2 + 3 + 4 + 5 which is 15.
