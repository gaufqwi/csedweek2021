Friday Trivia

Give these trivia questions a try. Click the blurry text to see the answers.

  1. What Stanford computer scientist literally wrote the book on computer algorithms: the multi-volume work in progress called The Art of Computer Programming? You might think the first letter of his last name is silent, but it isn't.

    Answer: Donald Knuth

    Knuth also wrote the TeX typesetting system that is still widely used for scientific and mathematical papers. If you find an error in one his books he will send you a check for $2.56.

  2. What object-oriented computer language was developed by Sun Microsystems in the mid 1990s? It is named for a beverage popular among programmers who work late into the night.

    Answer: Java

    Java was designed to compile into "bytecode" rather than machine code. That means that the same Java program can run on any computer - Mac, PC, Unix, etc. - without any changes.

  3. Bitcoin and other cryptocurrencies are based on a kind of database that stores data in an ordered list of records on many, many computers spread all around the world. What is the two word name of that type of database?

    Answer: block chain

    The trick to making a block chain system work is getting everyone to agree on the same data. Bitcoin does this by forcing anyone who wants to participate to solve a hard mathematical problem. That makes it hard to cheat the system as long a majority of people are playing fair.

  4. What number will the following Swift program print out? 
    
let numbers = [12, 5, 7, 15, 2, 9]
var sum = 0

for n in numbers {
    if n < 10 {
        sum = sum + n
    }
}
print(sum)

    Answer: 23

    The for loop repeats for each number in the list. Because of the if statement, only the numbers less than 10 get added to sum. So sum will equal 5 + 7 + 2 + 9, which is 23.
