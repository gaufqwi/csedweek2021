Tuesday Trivia

Give these trivia questions a try. Click the blurry text to see the answers.

  1. What 19th century woman is often considered to be the world's first computer programmer? She wrote a program to calculate a mathematical sequence called the Bernoulli numbers.

    Answer: Ada Lovelace

    The mechanical computer that Ada's program was intended to run on, Charles Babbage's Analytical engine, was never built, so she never got a chance to test her program. Still, she deserves major credit for her insights on how computers could be used as general purpose problem solving machines, not just calculators for arithmetic.

  2. What computer language invented by Guido van Rossum was named for a British comedy troupe?

    Answer: Python

    The comedy troupe was Monty Python, who had a TV program and made several movies. Today Python is one of the most popular computer languages around the world. It was designed to be easy to learn yet still powerful.

  3. What line of inexpensive, credit card sized computers popular with hobbyists is named for a fruit and a mathematical constant?

    Answer: Raspberry Pi

    Raspberry Pi computers aren't just for fun. In Brazil they are being used to run ventilators used by COVID patients.

  4. What string does the Javascript code below print? 
    
let fruits = ['apple', 'banana', 'grape', 'orange', 'pear'];
console.log(fruits[3]);

    Answer: 'orange'

    In Javascript, as in many computer languages, arrays are 0-based. That means that the first item has an index of 0, not 1. Thus in the given code fruit[0] is 'apple', fruit[1] is 'banana', fruit[2] is 'grape', and fruit[3] is 'orange'.
