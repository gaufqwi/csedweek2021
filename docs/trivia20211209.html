Thursday Trivia

Give these trivia questions a try. Click the blurry text to see the answers.

  1. What is the name of the Finnish-American computer scientist who created the open source Unix-like operating system that is at the core of many web servers, mobile phones, and other devices? He shares his name with both a Nobel prize-winning chemist and a blanket-carrying cartoon character.

    Answer: Linus Torvalds

    Torvalds started writing the Linux operating system in 1991 as a way to learn about the Intel 386 computer chip. He also wrote the Git version control system that used by developers all over the world.

  2. What common business oriented language was based on work done by Admiral Grace Hopper? Important during the 1960s and 1970s, this language is much less widely used today, though some mainframe systems depend n it.

    Answer: COBOL

    Grace Hopper is one of the most important pioneers in the field of computer science. Today (December 9) is her birthday.

  3. Many acronyms related to computer networking and the internet end with the letter P. Examples include IP, TCP, DHCP, and HTTP. What word does the letter P stand for in each of these acronyms?

    Answer: protocol

    A protocol is a common set of rules for interacting and communicating shared among a group of people (or computers). These protocols ensure that computers "speak the same language" when they exchange data over the network.

  4. In the complete C program shown below, which of the three functions would be the first (and in fact only) to be called when the program is run? 
    
#include <stdio.h>

init() {
   printf("Hello, World from init!");
}

start() {
   printf("Hello, World from start!");
}

main() {
   printf("Hello, World from main!");
}

    Answer: main()

    C, developed by Dennis Ritchie in the early 1970s, is the most popular programming language in the world. The main() function is the entry point for C programs.
