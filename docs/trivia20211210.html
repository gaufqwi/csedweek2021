Friday Trivia

Give these trivia questions a try. Click the blurry text to see the answers.

  1. What 18th century mathematician and engineer designed the Difference Engine and the Analytical engine, thought to be the first digital programmable computers?

    Answer: Charles Babbage

    Babbages "computers" were mechanical, not electronic. Though neither was constructed during his lifetime, since then working models have been built from his designs.

  2. What C-like language with a two letter name was designed at Google in the late 2000s?

    Answer: Go

    Notable projects written in Go include the Docker and Kubernetes container systems and CockroachDB, a scalable database.

  3. Broadly speaking, modern computer processors can be divided into RISC and CISC types. RISC stands for "reduced instruction-set computer". What does CISC stand for?

    Answer: complex instruction-set computer

    CISC processors generally support more powerful instructions than RISC processors, but each instruction might require more time or energy to execute, so it is really impossible to say that one type is always better than the other.

  4. What number will the following Swift program print out? 
    
let numbers = [12, 5, 7, 15, 2, 9]
var sum = 0

for n in numbers {
    if n*n > 50 {
        sum = sum + n
    }
}
print(sum)

    Answer: 36

    The for loop repeats for each number in the list. Because of the if statement, only the numbers whose square is greater than 50 get added to sum. So sum will equal 12 + 15 + 9, which is 36.
