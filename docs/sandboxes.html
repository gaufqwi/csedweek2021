Sandboxes for Programming

On the playground a sandbox is place to play where you can make your own fun. There is sand there that you can dig through and build with, and some walls to keep the sand in so you don't make a big mess, but that's pretty much it. You get to decide what you want to imagine and create in the sandbox.

In the world of programming a sandbox is pretty similar. The word actually can refer to a couple of different things, but for our purposes a sandbox is an environment where you can write and run code right in your web browser. That means you can get started coding without having to install anything on your computer. Sandboxes are the perfect way to learn about and experiment with different programming languages. They are also great if you just want to write some code quickly to solve a problem or a puzzle. More on programming puzzles later in the week.

Here are some of my favorite coding sandboxes for different languages. Some of these are designed for beginners and some are intended for use by more experienced programmers.

Block-based Programming
Scratch You may have encountered Scratch before through a guided Hour of Code activity, but the Scratch editor will let you create your projects from, well, scratch.
MIT App Inventor App Inventor is a block-based environment for creating and testing apps that run on phones and other mobile devices.
Snap! Snap! is a block-based environment designed for more experienced programmers. It has some features that Scratch does not.
Python
Python Sandbox The name says it all. A simple sandbox for text-based Python programs.
Code HS CodeHS provides several different Python sandbox environments. Some are purely text-based and others provide different kinds of graphics. Check out the examples and see which one works best for you.
Javascript
Khan Academy Khan Academy's Javascript sandbox is great for drawing and animation.
P5.js P5 is very similar to Khan Academy in that it provides lots of Javascript functions for creating graphics. It is more powerful and gives you more options, but the tradeoff is that it is a little harder to get started.
JSFiddle JSFiddle lets you use Javascript to do what the language was originally designed for, creating interactive web pages. That means that you will probably need to write HTML and maybe CSS code as well.
Java
Code HS Java is an important language because it is used in AP Computer Science courses (and for other reasons too). There aren't too many sandboxes for Java available, but CodeHS has a good one.
