Tuesday Trivia

Give these trivia questions a try. Click the blurry text to see the answers.

  1. What English computer scientist is considered the inventor of the World Wide Web?

    Answer: Tim Berners-Lee

    Berners-Lee invented the web in 1989. He wrote the first browser in 1990 while employed at CERN near Geneva, Switzerland. The first website, which was about the web itself, was published on 20 December 1990.

  2. What computer language is named for a 17th century French mathematician? This man also lends his name to a famous triangle that is important in mathematics.

    Answer: Pascal

    The Pascal programming language was designed by Niklaus Wirth. Popular in the 1970s, it has largely been replaced by C. The triangle mentioned is called Pascal's Triangle, named for mathematician Blaise Pascal.

  3. Almost all modern computer chips are made primarily from what chemical element with atomic number 14?

    Answer: Silicon

    Silicon is a semiconductor, meaning that it can transmit an electrical current, but not as easily as a metal. By adding small amounts of other elements to a silicon chip, engineers can change its properties to create switches and other types of circuits.

  4. What string does the Javascript code below print? 
    
console.log('3' + '5')

    Answer: 35

    '3' and '5' may look like the numbers 3 and 5, but because they are in quotes they are actually strings. In Javascript, the + operator applied to two strings concatenates them, i.e. joins them together. That is true even if the strings happen to look like numbers.
